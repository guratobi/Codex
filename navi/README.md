# 🧚 나비 (Navi)

곁에서 따라다니며 **때맞춰 콕 찔러주는** 개인 비서.
젤다 '시간의 오카리나'의 요정 나비에서 따왔다 — *"이봐! 들어봐!"*

떠오를 때 **텔레그램으로 한 줄** 던지면, 나비가 **집/회사로 분류**해 기억해 뒀다가
**그 장소에 맞는 때**(출근·퇴근 즈음, 또는 도착 순간)에 먼저 알려준다.
끝낸 일은 시각과 함께 보관되고, 모든 흐름이 **장부(git + log)** 로 남는다.

## 어떻게 굴러가나
```
폰(어디서나) ──텔레그램 한 줄──▶ 나비 @ 우분투박스(24시간)
                                  │ 분류(집/회사)·저장·done·로그
                                  ▼
                         비공개 brain 레포 (깃 = 장부)
                                  │
   회사서 CC 켜면 ◀─ SessionStart 훅이 pull+출력 ─┘
   집/회사 도착 즈음 ◀─ 나비가 텔레그램으로 그 목록 push
```

## 핵심 설계
- **외부 의존성 0** — 파이썬 표준 라이브러리만. 우분투 박스에서 `python3 navi.py` 면 끝.
- **자동 분류** — 키워드로 집/회사 판별, 애매하면 되물어본다.
- **이직/이력서/면접류는 집(개인) 쪽** — 회사 노트북 화면엔 절대 안 뜨게 의도적으로.
- **두 갈래 노출** — ① 텔레그램 시간대 푸시 ② 회사 CC SessionStart 훅. (옵션: GPS 도착 푸시)
- **장부** — 모든 추가/분류/완료가 `log.jsonl` + 깃 커밋으로 남는다.
- **자기소개 고정 공지** — `ABOUT.md` 를 텔레그램에 핀으로 고정, 내용 바뀌면 자동 갱신.
- **채팅 자동 청소** — 오래된 대화는 나비가 지운다(`NAVI_CHAT_TTL_HOURS`). 기록은 brain 에만 남고 채팅창은 가볍게.

## 구성
| 파일 | 역할 |
|---|---|
| `navi.py` | 나비 본체 (텔레그램 폴링·분류·로그·시간대 푸시·웹훅) |
| `navi.env.example` | 설정 템플릿 (`navi.env` 로 복사해 사용) |
| `setup.sh` | 검둥이에서 한 번에 설치/등록하는 도우미 |
| `ABOUT.md` | 나비 자기소개 (텔레그램 고정 공지로 게시됨) |
| `hooks/session-start.sh` | 회사 노트북 CC 세션 시작 시 회사 큐 출력 |
| `systemd/navi.service` | 박스에서 24시간 자동 실행 |
| `brain.example/` | brain 폴더 형식 예시 (실제 brain 은 별도 비공개 레포 권장) |
| `SETUP.md` | 한 단계씩 셋업 가이드 |

## 설치
**[SETUP.md](./SETUP.md)** 를 순서대로. 요약:
1. 봇 토큰(날씨 봇 재사용 OK) 2. 박스에 코드 3. 비공개 brain 레포
4. `navi.env` 작성 5. 테스트 실행 6. systemd 등록 7. 폰 뒷면 탭 8. 회사 CC 훅

## 쓰는 법 (텔레그램에서)
- 그냥 한 줄 → 자동 분류 / `회사: …`·`집: …` → 지정 추가
- `집`·`회사` → 목록 / `끝 a1b2` → 완료 / `목록`·`인박스` / `/help`

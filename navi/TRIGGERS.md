# 나비 도착 트리거 (집·회사 알림)

집/회사에 **도착하는 순간** 나비가 그곳 할 일을 텔레그램으로 쏴준다.
고정 시각이 아니라 "진짜 도착"이 신호라 — 시차근무·늦은 퇴근·집 비우는 날도 알아서 맞는다.

## 한눈에
| 어디 | 메인 알림 | 보너스 |
|---|---|---|
| 🏢 회사 | 💻 흰둥이서 CC 켜는 순간 큐 출력 (와이파이·GPS 불필요) | 📍 GPS 도착 → 폰 push |
| 🏠 집 | 📶 집 와이파이 붙는 순간 → 폰 push | 📍 GPS 도착 → 폰 push |

> **회사 와이파이는 안 써도 된다.** 회사 쪽은 CC 훅이 메인(흰둥이 로컬이라 제일 정확),
> 폰 알림을 더 원하면 GPS 도착으로 받으면 된다. 와이파이 트리거는 집만 쓴다.

```
폰: 도착/와이파이 ──(아이폰 자동화)──▶ 검둥이의 나비 웹훅 ──▶ 텔레그램 push
```

준비물: 검둥이(나비 상주), 아이폰, 무료 Tailscale 계정.

---

## 1. 검둥이: 웹훅 켜기
`navi.env` 에서:
```
NAVI_PUSH_WORK=off          # 시간대 푸시 끔 (도착 기반으로 가니까)
NAVI_PUSH_HOME=off
NAVI_WEBHOOK_PORT=8787
NAVI_WEBHOOK_SECRET=길고-아무도-모를-비밀문자열
```
재시작: `sudo systemctl restart navi`. 로그에 `도착 웹훅 :8787 대기 중` 뜨면 OK.

## 2. 폰 ↔ 검둥이 잇기 (Tailscale)
회사망 밖(집·셀룰러)에서도 폰이 검둥이에 닿게, 공개 노출 없이 사설로 잇는다.
1. 검둥이: `curl -fsSL https://tailscale.com/install.sh | sh && sudo tailscale up`
2. 아이폰: App Store 에서 **Tailscale** 설치 → **같은 계정**으로 로그인
3. 검둥이 주소 확인: 검둥이에서 `tailscale ip -4` → 예) `100.x.y.z`
   (이 주소가 폰에서 검둥이를 부르는 이름이 된다)

## 3. 아이폰 자동화
**단축어 앱 > 자동화 > + > 개인용 자동화 생성**

### 회사 도착 (GPS — 와이파이 안 씀)
- 트리거: **도착** → 위치를 회사로
- 동작: **"URL의 콘텐츠 가져오기"** 추가 →
  `http://100.x.y.z:8787/arrive?place=work&key=길고-아무도-모를-비밀문자열`
- **즉시 실행** 켜기(묻지 않기)

### 집 (집 와이파이 추천, 또는 도착)
- 트리거: **Wi‑Fi** → 집 와이파이 선택  (또는 **도착** → 위치=집)
- 동작 URL: `http://100.x.y.z:8787/arrive?place=home&key=길고-아무도-모를-비밀문자열`
- **즉시 실행** 켜기

## 4. 테스트
폰 사파리에 위 URL 을 직접 쳐봐 → 텔레그램에 그 목록이 오면 성공.
이제 집/회사 드나들 때마다 자동으로 온다.

---
### 막히면
- 텔레그램에 안 옴 → 검둥이 로그 `journalctl -u navi -f`, SECRET·포트·주소 확인
- URL 이 안 열림 → 폰 Tailscale 켜졌나, 검둥이 `tailscale status` 확인
- 회사가 Tailscale 자체를 막으면 → `cloudflared` 터널로 대안 가능 (말해줘, 가이드 줄게)

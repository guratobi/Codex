# 나비 도착 트리거 (집·회사 알림)

집에 도착하거나 회사에서 흰둥이를 켜는 순간, 나비가 그곳 할 일을 텔레그램으로 쏴준다.
고정 시각이 아니라 "진짜 도착"이 신호라 — 시차근무·늦은 퇴근·집 비우는 날도 알아서 맞는다.

## 한눈에
| 어디 | 메인 알림 | 보너스 |
|---|---|---|
| 🏢 회사 | 🔌 **흰둥이 켜지면** 검둥이가 감지해 폰으로 push (Tailscale) | 💻 CC 켤 때 화면 출력 · 📍 GPS 도착 |
| 🏠 집 | 📶 집 와이파이 붙는 순간 → 폰 push | 📍 GPS 도착 → 폰 push |

> **회사 와이파이는 안 써도 된다.** 회사 쪽은 **흰둥이가 켜지는 것**이 신호 —
> 검둥이가 Tailscale 로 흰둥이 상태를 지켜보다 켜지는 순간 회사 목록을 쏜다.
> (흰둥이엔 스크립트 안 깐다. Tailscale 앱만.) 와이파이 트리거는 집만 쓴다.

```
폰: 도착/와이파이 ──(아이폰 자동화)──▶ 검둥이의 나비 웹훅 ──▶ 텔레그램 push
```

준비물: 검둥이(나비 상주), 흰둥이(회사 노트북), 아이폰, 무료 Tailscale 계정.

---

## 1. 검둥이: 웹훅 켜기
`navi.env` 에서:
```
NAVI_PUSH_WORK=off          # 시간대 푸시 끔 (도착 기반으로 가니까)
NAVI_PUSH_HOME=off
NAVI_WEBHOOK_PORT=8787
NAVI_WEBHOOK_SECRET=길고-아무도-모를-비밀문자열
NAVI_WHITEY_HOST=100.x.y.z  # 흰둥이 Tailscale 주소(2번에서 확인). 켜지면 회사 알림
```
재시작: `sudo systemctl restart navi`. 로그에 `도착 웹훅 :8787 대기 중` 뜨면 OK.

## 2. Tailscale 로 묶기 (검둥이·흰둥이·폰)
공개 노출 없이 사설로 잇는다. 셋 다 **같은 계정**으로 로그인.
1. 검둥이: `curl -fsSL https://tailscale.com/install.sh | sh && sudo tailscale up`
2. 흰둥이: Tailscale 설치(앱 또는 같은 스크립트) → 로그인
3. 아이폰: App Store 에서 **Tailscale** 설치 → 로그인
4. 주소 확인: 검둥이·흰둥이에서 각각 `tailscale ip -4` (예 `100.x.y.z`)
   - **검둥이** 주소 → 폰 자동화 URL 에 쓰고
   - **흰둥이** 주소 → 위 `NAVI_WHITEY_HOST` 에 넣는다 (넣었으면 검둥이 나비 재시작)

## 3. 아이폰 자동화 (집은 필수, 회사는 보너스)
**단축어 앱 > 자동화 > + > 개인용 자동화 생성**

> 회사는 위 `NAVI_WHITEY_HOST` 덕에 **흰둥이만 켜면 자동**이라 폰 자동화가 꼭 필요친 않다.
> 폰으로도 더 일찍 받고 싶을 때만 아래 "회사 GPS"를 추가(선택).

### (선택) 회사 GPS 도착
- 트리거: **도착** → 위치를 회사로
- 동작: **"URL의 콘텐츠 가져오기"** →
  `http://검둥이주소:8787/arrive?place=work&key=길고-아무도-모를-비밀문자열`
- **즉시 실행** 켜기(묻지 않기)

### 집 (집 와이파이 추천, 또는 도착)
- 트리거: **Wi‑Fi** → 집 와이파이 선택  (또는 **도착** → 위치=집)
- 동작 URL: `http://100.x.y.z:8787/arrive?place=home&key=길고-아무도-모를-비밀문자열`
- **즉시 실행** 켜기
- **집 와이파이가 2개면** 자동화를 2개 만들어(각 와이파이마다) 같은 URL 로. 둘 다 걸어도
  나비가 중복은 무시한다(기본 10분 쿨다운, `NAVI_ARRIVE_COOLDOWN_MIN` 로 조절).

## 4. 테스트
폰 사파리에 위 URL 을 직접 쳐봐 → 텔레그램에 그 목록이 오면 성공.
이제 집/회사 드나들 때마다 자동으로 온다.

---
### 막히면
- 텔레그램에 안 옴 → 검둥이 로그 `journalctl -u navi -f`, SECRET·포트·주소 확인
- URL 이 안 열림 → 폰 Tailscale 켜졌나, 검둥이 `tailscale status` 확인
- 회사가 Tailscale 자체를 막으면 → `cloudflared` 터널로 대안 가능 (말해줘, 가이드 줄게)
